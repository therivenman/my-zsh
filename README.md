my-zsh
========

My Zsh configuration files (requires oh-my-zsh)

Installation:

1. Clone repo:

  `git clone https://github.com/therivenman/my-zsh.git ~/.my-zsh`

2. Install `oh-my-zsh`:

  https://github.com/robbyrussell/oh-my-zsh

3. Install `.zshrc`:

  ```
  rm ~/.zshrc
  ln -s ~/.my-zsh/zshrc ~/.zshrc
  ```
